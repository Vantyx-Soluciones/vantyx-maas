# Bienvenido a Vantyx MaaS (Module as a Service)

Gracias por elegir la solución de facturación electrónica de Vantyx Soluciones. Al usar este modelo, usted cuenta con la potencia de nuestra infraestructura de nube para su gestión empresarial.

## Contenido del Paquete

- `vantyxmaasconnector_1.0.0.zip`: Módulo para instalar en su Dolibarr.
- `legal/`: Contratos y descargos de responsabilidad que rigen este servicio.

## Instrucciones de Instalación

1. Descomprima `vantyxmaasconnector_1.0.0.zip` en la carpeta `custom/` de su instalación de Dolibarr.
2. Active el módulo en **Configuración -> Módulos -> Conector Vantyx MaaS**.
3. Ingrese el Token de activación que le proporcionamos en la configuración del conector.
4. Active el módulo nativo de **Webhooks** en Dolibarr y configure la URL de destino según las instrucciones en la pantalla de configuración del conector.
